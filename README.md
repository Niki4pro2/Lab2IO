Изучение симплекс-метода для решения задач линейного программирования с использованием симплекс-таблиц, получение навыков кодирования изученного алгоритма, отладки и тестирования соответствующих программ.

## 📋 Задания

### 🔹 Задание 1

Выяснить:

* какой вид должна иметь задача ЛП для применения симплекс-метода в чистом виде;
* как составляется первая симплекс-таблица.

---

### 🔹 Задание 2

Изучить алгоритм перехода от одной симплекс-таблицы к другой при решении задачи симплекс-методом.

---

### 🔹 Задание 3

Запрограммировать и отладить изученный алгоритм.

✔ Выполнено:

* Проведено тестирование программы
* Получены результаты вычислений

---

### 🔹 Задание 4

Решить задачу вручную (вариант №8).

#### Исходная задача:

```
z = x1 + 3x2 − 5x4 → max

3x1 + 4x2 + x3 + 2x4 = 27
4x1 + 5x2 − 3x4 + x5 = 32
5x1 − 2x2 + 8x4 + x6 = 24

xi ≥ 0 (i = 1..6)
```
